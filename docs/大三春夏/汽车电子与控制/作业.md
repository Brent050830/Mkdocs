- 被控对象的建模（按油膜流量考虑：）
	- 直接进入气缸的燃油流： $\dot{m}_{f1} = (1-X)\dot{m}_{fi}$
	- 油膜蒸发进入气缸的燃油流： $\dot{m}_{f2} = \frac{1}{\tau} m_{ff}$
	- 油膜质量的动态变化率： $\frac{dm_{ff}}{dt} = X\dot{m}_{fi} - \dot{m}_{f2}$
	- 实际进入气缸的总燃油： $\dot{m}_{f} = \dot{m}_{f1} + \dot{m}_{f2}$

> $X$：燃油吸附的比例  
> $m_{fi}$：喷油器喷射的燃油的质量  
> $m_{ff}$：进气道湿壁油膜的燃油质量

**补偿器**的目的是计算出应该给喷油器的指令 $\dot{m}_{fi}$，以抵消物理模型的延迟。

- 控制器的建模：补偿后喷油指令： $\dot{m}_{fi} = \frac{1}{1-X}(\dot{m}_{fu} - \dot{m}_{fv\_comp})$  
	此处的 $\dot{m}_{fv\_comp}$ 计算公式为：${\frac{\mathrm{d}m_{\mathrm{ff}}}{\mathrm{d}t}}={\frac{1}{\tau}}(-m_{\mathrm{ff}})+X\dot m_{\mathrm{fi}}$  
	注： $\dot{m}_{fu}$ 为目标燃油量，$\dot{m}_{fv\_comp}$ 为补偿器内部根据当前状态估算的油膜蒸发量，估算公式与上述物理过程的方程完全一致（但是这里的油膜蒸发量是需要预测的，而不是实际计算的（因为算不出来）），由采取喷油量之后实际的喷油量使用方程计算出来的，与真实的相比会滞后的。

#### 3. 
1. 参数 $X$：燃油附壁比例（吸附比）

* **物理意义：** $X$ 代表喷油器喷出的燃油中，**直接撞击并吸附在进气道壁面上形成油膜的质量分数** 。剩下的 $(1 - X)$ 则是直接以气态或微小液滴形式随空气进入气缸的燃油比例。
* **对动态传输的影响（决定“瞬态落差”）：** 在稳态工况下，由于油膜的“蒸发量”等于“新附壁量”，$X$ 的大小不影响最终进入气缸的总油量。
    * 在**瞬态工况**（如猛踩油门，喷油量呈阶跃增加）发生的第一瞬间（$t=0$），进入气缸的燃油量会瞬间跳跃到 $(1 - X)$ 倍的指令油量 。
    * **结论：** $X$ 的值越大（表明越多的油被墙壁“截留”），瞬态发生时的**初始燃油缺口就越大**。在驾驶感受上，这就意味着加速初期混合气会严重偏稀，导致发动机响应迟滞或抖动。

2. 参数 $\tau$：油膜蒸发时间常数

* **物理意义：** $\tau$ 代表附壁油膜通过传热和强制对流，**蒸发进入进气气流的快慢程度**。它主要受进气温度、进气流量（气流速度）和燃油自身饱和蒸汽压的影响 。
* **对动态传输的影响（决定“恢复速度”）：**
    * 在产生初始的燃油缺口（由 $X$ 决定）之后，油膜会随着时间逐渐积累并增加蒸发量，最终使实际进缸油量追上目标喷油量。
    * 公式中的 $e^{-t/\tau}$ 是一个指数衰减项 。$\tau$ 越小，指数衰减得越快，实际油量逼近目标值的速度就越快。
    * **结论：** $\tau$ 决定了燃油缺口**持续的时间长短**。在冷启动时（进气道温度低），$\tau$ 会非常大，导致空燃比偏离目标值的时间大幅拉长；而在发动机热机且高转速时，$\tau$ 较小，动态响应会迅速恢复。
