### 传感器类型
- 视觉类传感器：摄像头（可以安装在车辆上，也可以固定机位）  
	语义分割摄像头可以识别车辆还是行人
- 距离与几何类传感器  
	激光雷达  
	毫米波雷达
- 导航与运动传感器
- 逻辑与事件类传感器

#### 视觉类传感器
传感器同步：务必将世界设置为**同步模式**，这样可以确保传感器数据、物理步进和图像帧号严格对其，不会出现偏移

为了输出视频，我们需要安装 `pip install opencv-python`（关代理）

#### 距离与几何类传感器
