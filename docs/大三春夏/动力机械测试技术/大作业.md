## 流速传感器
测量**发动机进气流速**和测量普通风速其实不太一样，因为进气流速往往：

* 流速范围大（从低到高几十 m/s）
* 流场湍流厉害
* 空气密度变化（温度 / 进气压力）

所以要选**适合高流速、动态响应快、输出可 DAQ 采集的传感器**。下面是详细推荐：

---

## ✅ 推荐传感器类型

### ⭐ 1. 皮托管 + 差压传感器（最常见、最稳妥）

➡ 这类是汽车发动机进气流速最常用的方案

**原理：**

皮托管测量动压 + 静压：

$$  
\Delta P = P_{\text{动}} - P_{\text{静}}  
$$

根据空气动力学：

$$  
v = \sqrt{\frac{2 \Delta P}{\rho}}  
$$

其中：

* (v)：流速（m/s）
* (\Delta P)：压差
* (\rho)：空气密度（≈1.1~1.2 kg/m³）

---

### 插件 + 传感器选择

| 部件    | 建议型号 / 类型        | 输出                   |
| ----- | ---------------- | -------------------- |
| 皮托管   | 小气管皮托管（汽车进气管适配）  | 动/静压                 |
| 差压传感器 | ±500Pa / ±2500Pa | 0~5V / 4~20mA（需电阻转换） |

推荐常用压差模块（输出电压）：

✔ SDP600 / SDP610 差压传感器  
✔ MPX5500 / MPXV7002DP（汽车流速常用）

这些输出都是模拟电压，适合 NI 采集。

---

## ✅ DAQ 读入方式

你之前选的是：

```text
模拟输入 → 电压 - 连续输入
```

进气流速测量也是用这个。  
因为压差传感器输出电压对应 ΔP。

---

## 📌 LabVIEW 里的计算流程

### ① DAQmx 采集 ΔP 信号

```text
DAQmx Read → 波形 → Y 数组 (压差电压)
```

### ② 把电压转成压差值

例如：

传感器输出范围 0~5V 对应 ΔP -50~+50Pa：

$$  
\Delta P = (V - 2.5) / 5.0 × 100Pa  
$$

换算公式根据传感器说明书来。

---

### ③ 压差转换成流速

$$  
v = \sqrt{\frac{2 \Delta P}{\rho}}  
$$

注意：

* ΔP < 0 时表示逆流（可根据需要处理）
* 空气密度 ρ 可取常数，也可以用于更精准计算：

$$  
\rho = \frac{P_{\text{进气}}}{R \cdot T}  
$$

其中：

* (P_{\text{进气}})：进气压力
* (T)：进气温度
* (R = 287)

如果你已经有温度/压力信号，可以进一步修正密度。

---

## 📊 LabVIEW 示例框图应该长这样

```python
DAQmx Read (差压电压) ——→ Y数组 ——→ 平均/滤波
                                    ↓
                          电压 -> 压差 (Pa) (线性换算)
                                    ↓
                         差压 -> 速度 (sqrt 公式)
                                    ↓
                          流速显示 / 波形图表
```

---

## 📌 为什么选皮托管而不是热式传感器

| 方案              | 优点                  | 缺点            |
| --------------- | ------------------- | ------------- |
| 热式风速            | 响应快                 | 需要大量标定、受温度影响大 |
| 转子 / 叶轮风速       | 对进气管道不方便装           | 机械磨损、量程受限     |
| **皮托管 + 压差传感器** | **适合高速流、车用常用、输出稳定** | 需要安装管路        |

---

## 📌 DAQmx 配置注意事项

✔ 采样率按发动机转速设定（一般 500~2000Hz 足够）

✔ 波形格式：

选择：

```python
模拟输入 → 电压 - 连续输入
```

而不是：

```python
热电偶 / RTD / 桥式
```

因为皮托管输出的是电压，不是热敏或电阻型信号。

---

## 📌 参考数值示例

假设传感器标定为：

```python
0~5V 对应 ΔP 0~500 Pa
```

则：

```python
ΔP = V / 5 * 500
```

空气密度取 1.2：

```python
v = sqrt(2*ΔP / 1.2)
```

如果 V = 2.5V：

```python
ΔP = 2.5/5*500 = 250 Pa
v = sqrt(500/1.2) ≈ 20.4 m/s
```
