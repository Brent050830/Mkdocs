## 程序结构 2
### 条件结构
if else

- 前面板中找到布尔——开关按钮
- 在程序框图中找到结构——条件结构
	- 将这个布尔开关的输出给到条件结构的输入
	- 条件结构中间上面有真和假的分支
- 前面板中找到字符串的显示控件，给他一个字符串的常量
- 切换到假分支中，对字符串的显示控件创建一个值，并仅仅输入
- 再加一个 while 循环，就会得到这样的程序：  
	![](png/Pasted%20image%2020260523180550.png)

---
条件结构的输入还可以是**整形和字符串**类型（此时就相当于 switch case 了）

- 前面板——数值输入控件
	- 这个默认是输入小数的，可以右键，表示法，选择整形（int）
	- 然后就变蓝了
- 然后将这个输入控件与条件语句相连，上面的就变成 0,1 了
- 现在自动创建了两个分支（相当于一个 if else）
- ![](png/Pasted%20image%2020260523181311.png)
	- 现在这个就代表输入是 1 的时候输出为我是 1
	- 输入不是 1 的时候到默认那里，输出为我不是 1
- 想多几个条件的话（在前后添加分支，现在就相当于 switch case 了）  
	默认分支是必须有的  
	设置为其余的话，可以先将之前的默认的右键取消掉，设置为新的

字符串类型也是相似的，就是 switch case  
![](png/Pasted%20image%2020260523182138.png)

## 蔟
将多种类型的数据集成在一个数据结构中

### 创建
同样的，找到蔟，然后往里面放元素
