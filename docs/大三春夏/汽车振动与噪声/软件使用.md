## 使用
### 新建项目
点击 Analysis，新建项目  
![](png/Pasted%20image%2020260415210116.png)  
所以点击 Pool project 即可

### 流程介绍
![](png/Pasted%20image%2020260415210242.png)

下面就对这几部分分别说明一下：

### 数据源池
在导航栏中将所需的数据拖入我们的数据源池中（.hdf、.dat、.atfx 这三种格式的数据可以直接插入（Insert））  
![](png/Pasted%20image%2020260415210503.png)

- 数据插入数据源池之后，点击三角图标，可以展开数据的通道  
	![](png/Pasted%20image%2020260415210620.png)  
	可以看到数据的通道数量、名称、通道名、单位、采样频率等等信息
- 右键所选的数据，可以进行通道名的编辑等等  

### 滤波池
在滤波池右键—Insert—General可以插入积分、微分、单位转换、重采样等
![](png/Pasted%20image%2020260415211740.png)
