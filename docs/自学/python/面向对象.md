## 初识对象
组织数据的形式

使用变量记录数据太慢了  
在程序中设计表格，称之为类：

#### 类的初步

```python
class Student:
    name = None # 记录姓名
    gender = None # 记录性别
    nationality = None # 记录国籍
    native_place = None # 记录籍贯
    age = None # 记录年龄
```

通过这种方式建立了一个表格

---
之后创建对象：

```python
# 创建一个对象（类比生活中，打印一张登记表）
stu_1=Student() # 创建了一个学生对象，赋值给stu_1,既是变量，也是对象
stu_2=Student() # 创建了一个学生对象，赋值给stu_2
```

然后填写表格（赋值）

```python
# 给对象的属性赋值（类比生活中，填写登记表）
stu_1.name = "chen"
stu_2.name = "bo"
stu_1.gender = "male"
stu_2.gender = "male"
stu_1.nationality = "Chinese"
stu_2.nationality = "Chinese"
stu_1.native_place = "Beijing"
stu_2.native_place = "Shanghai"
stu_1.age = 20
stu_2.age = 22
```

输出的结果：

```text
chen
bo
male
male
Chinese
Chinese
Beijing
Shanghai
20
22
```

- 设计表格：设计类
- 大英表格：创建对象
- 填写表格：数据的属性赋值  
	就不需要多次创建变量（属性）了

## 类的成员方法
### 类的定义和使用

```python
class 类名称：
	类属性：就是定义在类中的变量（不需要在外面再多次定义了）
	类行为：类的行为：定义在了类中的函数（成员方法）
```

创建类对象：

```python
对象 = 类名称（）
```

---
- 类中定义的属性（变量）：成员**变量**
- 类中定义的行为（函数）：成员**方法**

#### 方法定义

```python
def 方法名（self， 形参1……）
	方法体
```

- self：（必须有）  
	用来表示类对象自身  
	在成员方法的内部，想要调用成员变量时，必须使用 self

```python
class Student:
    name = None # 记录姓名
    gender = None # 记录性别
    nationality = None # 记录国籍
    native_place = None # 记录籍贯
    age = None # 记录年龄

    def say_hi(self):
        print(f"Hi, I am {self.name}.")
        
# 调用对象的方法
stu_1.say_hi() # 输出：Hi, I am chen.
stu_2.say_hi() # 输出：Hi, I am bo.
```

输出为：`Hi, I am chen.  Hi, I am bo.`  
调用这个方法的时候，不需要写 self（当做透明的）

接收其他参数传入：

```python
class Student:
    name = None # 记录姓名
    gender = None # 记录性别
    nationality = None # 记录国籍
    native_place = None # 记录籍贯
    age = None # 记录年龄

    def say_hi(self):
        print(f"Hi, I am {self.name}.")
    def say_hi2(self,msg):
        print(f"Hi, I am {self.name}. {msg}")

stu_1.say_hi2("Nice to meet you!") # 输出：Hi, I am chen. Nice to meet you!
stu_2.say_hi2("哎呦不错呦") # 输出：Hi, I am bo. 哎呦不错呦
```

可见 `self` 定义的时候需要写，使用的时候不需要

> 核心就是设计类，基于类创建对象，使用对象实现工作

```python
import time

class clock:
    id = None
    price = None

    def ring(self, time):
        import winsound
        winsound.Beep(2000, time) # 发出1000Hz的声音，持续time毫秒
    
# 构建2个闹钟对象工作

clock_1 = clock()
clock_2 = clock()

clock_1.id = "003032"
clock_1.price = 19.9
print(clock_1.id) # 输出：003032
print(clock_1.price) # 输出：19.9
clock_1.ring(3000) # 发出声音

time.sleep(1) # 等待1秒

clock_2.id = "003033"
clock_2.price = 29.9
print(clock_2.id) # 输出：003033
print(clock_2.price) # 输出：29.9
clock_2.ring(1000) # 发出声音
```

> - 设计类

## 构造方法
`__init__()` 的方法进行构造方法（前后都是两个下划线）

- 构建类时传入的参数会自动提供给 __init__ 方法
- 构建类时，__init __ 方法会自动执行

```python
class Student:
    name = None # 记录姓名
    gender = None # 记录性别
    nationality = None # 记录国籍
    native_place = None # 记录籍贯
    age = None # 记录年龄

    def __init__(self, name, gender, nationality, native_place, age):
        self.name = name
        self.gender = gender
        self.nationality = nationality
        self.native_place = native_place
        self.age = age

stu_1 = Student("chen", "male", "Chinese", "Beijing", 20)

print(stu_1.name) # 输出：chen
```

## 内置方法
### 魔术方法
例如 `__init__` 这些类内置的方法

#### 字符串方法
`__str__` : 将对象转换为字符串

```python
class Student:
    def __init__(self, name, age):
        self.name = name
        self.age = age

stu_1 = Student("chen", 20)

print(stu_1)
print(str(stu_1))
```

这样写只能返回内存字符串

所以加上字符串的这个

```python
class Student:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def __str__(self):
        return f"Student(name={self.name}, age={self.age})"

stu_1 = Student("chen", 20)

print(stu_1)
print(str(stu_1))
```

输出为：

> Student(name=chen, age=20)  
> Student(name=chen, age=20)

#### 小于符号比较法
`__lt__` 方法

```python
class Student:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def __str__(self):
        return f"Student(name={self.name}, age={self.age})"

    def __lt__(self,other):
        return self.age < other.age # other为其他的对象
    
stu_1 = Student("chen", 20)
stu_2 = Student("bo", 22)


print(stu_1 < stu_2) # 输出：True
```

`__le__` 方法就支持比较>=和<=了  
`__eq__`：相等的判断

## 封装
将现实事物的属性和行为**封装**为程序的成员变量和成员方法

对用户隐藏的属性和行为  
类中提供了私有成员的形式来支持这种

**两个下划线开头命名**的就是私有成员了

```python
class Phone:

    __current_voltege = none # 私有属性，记录当前电压

    def __keep_single_core(self):
        print("保持单核运行") # 私有方法，保持单核运行

phone_1 = Phone()

# 访问私有属性和方法
# print(phone_1.__current_voltege) # 报错：AttributeError: 'Phone' object has no attribute '__current_voltege'
# phone_1.__keep_single_core() # 报错：AttributeError: 'Phone
```

可以看出是无法调用的

那么这些功能是给类中的其他成员使用的

```python
class Phone:

    __current_voltege = 7 # 私有属性，记录当前电压

    def __keep_single_core(self):
        print("保持单核运行") # 私有方法，保持单核运行

    def call_5G(self):
        if self.__current_voltege > 5:
            print("使用5G网络")
        else:
            self.__keep_single_core() # 调用私有方法，保持单核运行
            print("使用单核的方式运行，无法使用5G网络")
phone_1 = Phone()

# 访问私有属性和方法
# print(phone_1.__current_voltege) # 报错：AttributeError: 'Phone' object has no attribute '__current_voltege'
# phone_1.__keep_single_core() # 报错：AttributeError: 'Phone

phone_1.call_5G() 
```

也就是内部的成员（类内部成员方法）可以使用这些私有的变量和方法
