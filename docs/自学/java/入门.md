## 入门篇  
### 第一个代码  
所有的代码都必须在：  
![](png/Pasted%20image%2020260307215626.png)  
在这样的目录下，写的代码必须是：  
```java
package test1;# 名称与文件夹名称想通过

public class hello {# 函数名与文件名称相同
    public static void main(String[] args) {
    System.out.println("hello");
    }
}
```

> [!tip]+ 注意  
> 包的命名规则必须符合**变量的命名规则**  
> 这里包的名称必须和**文件夹的名称**相同  
> 函数名必须与文件的名称相同

#### class  
class 的作用是定义一个类  
```java
public class hello {

    }
```

`hello` 为类名  
大括号中间的部分为类的范围（一个函数吗）  

---
新建类的时候：  

- 可以直接右键文件夹，点击新建类：
```Java
package test1;

public class aaa {

}
```

> [!tip]+ 新的代码：
> 
> ```java
> package test1;
> // packge就是一个包：类定义在哪个包下
> public class aaa {
> //表示定义一个类
> // class：定义一个类，类是java中最基本的组成单元
> // aaa：类的名字
> // {}：表示类的范围，所有与的代码都需要写在这个大括号中
>     public static void main(String[] args) {
>     // 表示主程序的入口
>         System.out.println("hello world");
>         System.out.println("111");
>         // 上述的函数表示输出
>     }
> }
> 
> ```  

所以就相当于前面就是包和类（目录），直到 `public static void main(String[] args)` 才正式进入我们的代码

### 变量  
- 定义：是用来储存某个数据的**容器**
- 存储会发生变化的数据  
    就是一个**存储数据的小空间**，而不是里面的数据，只不过我们可以调用变量中的数据而已

---
格式：`int a=10;` （数据类型，变量名，幅值，数据值 ）  
![](png/Pasted%20image%2020260307225309.png)  

这个和 C 语言是差不多的（严格限制变量类型的）  

#### 练习  
```java
package zimianliang;

public class bianliang {
    // 余额和
    public static void main(String[] args) {
        // 1. 定义变量记录余额
        double a=100;
        double b=10;
        double c=20;
        // 计算总的
        System.out.println(a+b+c);
        // 2. 微信中最后的余额是多少
        a=a+10-2;
        System.out.println(a);
    }
}

```

#### 变量注意事项  
- 只能存储一个值
- 变量名不允许重复定义
- 变量在使用前一定要进行复制
- 一条语句可以定义多个变量，也可以连续赋值
